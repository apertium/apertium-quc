## Apostrophes

Some common Unicode encodings for apostrophe:

* `ʼ`: `U+02BC MODIFIER LETTER APOSTROPHE` ✔
* `’`: `U+2019 RIGHT SINGLE QUOTATION MARK`
* `'`: `U+0027 APOSTROPHE`
